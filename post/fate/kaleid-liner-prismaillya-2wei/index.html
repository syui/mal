Illyaが魔法の女の子になったのは運命ですか？ 多分。 彼女がミユに会うのを誘ったのは運命でしたか？ 多分。 Illyaは存在していた彼女の別の側面があるかどうか常に疑問に思ったのですか？ おそらくそうではありません。 ああ待って、あなたはおそらく誰か、または彼女の他の側が何であるか疑問に思います。 しかし、心配はありません。なぜなら、運命の宇宙のこのスピンオフで何かが可能であるからです。

人気のFate / Stay Nightフランチャイズのスピンオフとして、Fate / Kaleidが再び同じ名前の漫画を取り入れるための続編に戻ってくる。 シーズン1のように、それは魔法の少女の楽しみの10のエピソードから成っている。 Illyaの場合、ここの「楽しい」部分はおそらく強調されていません。 むしろ、それは主にファンのためです。 これは、Illyaの人生は、彼女が彼女が変わった自我を持っていることを発見した日の後には何の楽しいものでもないことが判明したからです（Kuroe von Einzbernとして知られています）。 またはKuroを要する。

「存在」によって、注目を集めることになると、黒は本当にケーキを取る。 シーズンは彼女のクラスメイトに恥ずかしがり屋のように走っている。 しかし、このショーは、それが真剣に、そして同じ時間ではないという事実を明らかにする。 この代替宇宙では、私たちは魔法の女の子を取得します。 最初のFate / Stay Nightには、召使とマスターがいます。 あなたは2つについて考えるとき、それは気分になると大きなギャップがあります。 しかし、今シーズンは何とかKuroが重要な役割を担っているというメッセージを伝えることができます。 これは、季節の後半には正当化されます。なぜなら、クロは、さらに大きな脅威に直面しているので、彼女の違いを脇に置くことができます。 問題は、他人が最初にIllyaがその行為をしていると信じているということです。 そしてそれをするために、奇妙な過活動派の女の子はいたずらに彼女のマナを排水するトリックスターの役割を果たす。 おそらくこれに適した用語は「キス」です。 シーズン1がユリ・アンダートーンを持っていると思ったら、シーズン2は本当にそのエンジンを加速します。 Kuroが無数のキャラクターにキスをするように、今シーズンのユーリトーンは屋根を突破する。 ある人はショック状態にあり、他の人は恥ずかしい、おそらく人生のために傷ついている。 さて、これは過言ではないかもしれませんが、ショーはその要点になります。 これは後で明らかにされる理由によるものです。

Fate / Kaleidを見たり、理解したり、評価したりする最良の方法は、おそらくストーリーではありません。 それは運命のような文字を一緒に結びつける方法で書かれています。 ですから、KuroとIllyaも反対の性格を持っていても似ています。 邪悪な双子ではなく、イロイアが魔法の女の子として彼女の役割を理解することができる誰かとしてクロを解釈することができます。 続編として、クラスカードのキャプチャは、黒の出現につながった。 そして、それは学校での恥ずかしい瞬間、魔法の戦い、そして人格関係の強化で引き起こされたトラブルからです。 これは、シーズン1と同様のキャラクターの関係に焦点を当てたショーのポイントに戻ります。MiyuとIllyaは、お互いに理解してから素晴らしい友達です。 黒にとって、彼女が人間ではない、あるいは本当の友達がいると考えて、彼女の性格を理解するのは難しいです。 それは、彼女が彼女の世界で受け入れられていないと感じて以来、やはり始まりです。 彼女は自分のやり方で受け入れを探しています。 当然のことながら、クロは普通のやりたいことがあるようです。 学校や家での経験は、彼女が宝のように覚えているように、貴重です。

戦闘シーンはほとんどの部分にとって魅力的ですが、壮大なカテゴリには及ばない。 これはおそらく、ショーのギミックとユリ・フェナニガンの愚かさによって損なわれるでしょう。 それを楽しむ意欲は、元の運命シリーズの聖なる殻のように感じるかもしれません。 ありがたいことに、少なくとも適応の基準による一貫性と一致します。 振り返ってみると、ショーに出てくる期待は、楽しみとエンターテインメントのようにもっと集計されるべきです。 それ以外の場合は、シーズン全体が不便を感じるかもしれません。 それを踏まえて、ショーの戦いは魅力的なOSTで流動的かつ意味深いものになる可能性がある。 KuroはArcherの戦闘服の改造版を着用し、戦士として際立っています。 この魔法と正常性の領域では、Illyaの典型的な学校生活と魔法の女の子としての役割の2つのバランスが取れます。 色は私には驚きましたが、そのショーは、そのぎこちなさには驚くほど魅力的ではありません。 しかし、これはまた、ファンサービスでそれ自身をオーバーライドするときに崩壊する。 最初のシーズンにはこの部分がありますが、続編はバスシーンやユリのキスなど、エッチファンサービスの多くを大きく取り入れています。 巨大なコメディは、繰り返すことができ、見ることができないときどきのマナーでそれ自身を誇張する。 それを要約すると、Fate / Kaleidの脳のないコメディは、釘を視聴者の忍耐に当てるハンマーのようなものです。

シルバーリンクは視覚的な性質を適応させる。 同様に、アニメーションスタイルには大きな変化はありません。 Illyaのキャラクターデザインは1日以上は成熟せず、他のキャラクターはほぼ同じままです。 おそらく黒は、彼女の衣装とIllyaに似た外観の主な魅力です。 いくつかのいたずらな目と日焼けした肌のアーチャーの修正バージョン。 さもなければ、シーズンはホットドッグの上にケチャップを注ぐようなファンサービスで満ちている。 最終的に、それは味がセットアップしようとするものを追い越すときに、濁って陳腐化してしまいます。 聴衆に「アピール」しようとすると、議論の余地のあるカメラの角度や水が水から吹き飛ばされることを期待してください。 一方、宮の戦闘衣装や戦闘機などの創造性があります。

サウンドトラックはまともであり、この適応の補足として役立つようになります。 それは栄光も恐ろしいものでもありません。 しかし、サウンドトラックは、運命のファンのための経験を向上させるものです。 Kuroのテーマと軽快なバックグラウンドサウンドトラック用の修正OSTは、このスピンオフの意味を与えます。 しかし、キャラクターの声の大部分は、幼稚な音と耳に刺すような音です。 今季、いたずらな声の様子を披露してくれたブローアウト・スターとして、再び黒に賞賛しなければなりません。 彼女の個性と声は、彼女の役割と完全に一致します。 賛美の信憑性は、OPとEDの歌にも訴えを付けることができます。 それはほとんどのものがまだ一般的な属性で覆われているので、最後は特別なものではありません。 Fate / Kaleidは魔法の女の子についてのショーかもしれませんが、サウンドトラックに関しては魔法のように推測するものはありません。

Fate / Kaleidにはたくさんの驚きはありません。 予測可能なストーリーとジェネリックスタイルにより、視聴者はなぜこの続編に従うのか不思議に思うかもしれません。 しかし、これを知ってください：ショー自体は、物語が実際に行くときを除いて、心をあまりにも真剣に取るために満たされていません。 結局のところ、それは運命の宇宙のファンのためのより軽快なスピンオフです。 あなたがファンでなくても、彼らの役割を楽しむことができるキャラクターがあります。 しかし、反復的なコメディ、穏やかな対話、不適切なマッシュアップが乱雑であるという前提に注意してください。 このショーは、Illyaの人生が彼女が魔法の世界として発展した場所からどのように拡大していくのか、ファンが期待したいと思うかもしれないものにもっと専念しています。 ショーがまだ生と小説をミックスに統合しているので、それはただの運命だとは言いません。

存在は奇妙な話題です。 どの年齢の人々にとっても、「あるべき」という意味、または、自分が誰であるかをどのように知っているかを、実際には真実に留意することは稀です。 しかし、いくつかの魔法の女の子といくつかの恥知らずを投げ込むとどうなりますか？ 運命/ kaleidライナープリズマ☆Illya 2wei！ （それは長くなった！）このアイデアをある程度探求し、前のシーズンと同じ品質を維持している。

ストーリー

F / k2（この時点で再び短縮！）は、F / kが中止されたところから始まります。 IllyaとMiyuはRinとLuviaによって再び魔法の事件で彼らを助けるために募集されます。 しかし、彼らの最初の遠足では、間違いなく何かが起こります。

F / kが主にIllyaとMiyuの間のキャラクターのダイナミクスに焦点を当てた場合、今回はフォーカスが関係から存在のアイデアに移ります。 「私はそうだと思う」というフレーズは即座に気になります。 しかし、新しく登場したキャラクターのクロは、この単純な考え方では十分ではありません。 未知の起源、不利な目標、そして彼女の奇妙な行動では、彼女の存在の問題は答えにくいものになります。 ほとんどの場合、ショーは、これらのアイデアの結論を少なくとも理解できるようにするために、その啓示とキャラクタリゼーションをペーシングするといううまい仕事をしています。 これは何か「次のレベル」ではないが、ショーのためにはうまくいったと思った。

ショーが哲学的な領域に向かって進まないとき、それは多かれ少なかれ、人生ショーの一部です。 それは、ドッジボールやベーキングパンの再生など、毎日の通常のアクティビティを行うキャストを特長としています。 これらのセグメントは、コメディーとキャラクターの相互作用をうまく強調し、前のシーズンからの全体的な改善と思われました。

ショーの最強の部分の1つは最終的なエピソードです。 F / kにはさまざまな興味深い戦いがあり、最後にF / k2がこれらのルーツに戻ります。 戦いと並んで、ショーは次のシーズンに何が起こるのかをうまく設定します。 この最後のいくつかの戦いの場面を見て、ショーのバランスがとれた展覧会、生活のスライス、そして戦いがいかにうまくいったかが明確になりました。 1つの部分が他の部分よりも重大なことは決してありません。

アニメーション

F / kと同様に、F / k2は高い品質を維持します。

芸術のスタイルは比較的同じです。 文字は光沢があり、背景はきれいで、照明はこれまで以上に詳細に見えます。 このショーでは、輝き、フラッシュ、輝きを引き続き使用し、戦闘やキャラクターの行動に余計な「オイム」を与えることができます。

実際のアニメーションはもう一度かなり良いです。 戦いは素晴らしいディレクションを特徴とし、素敵な振り付けと「カメラ」アングルの使い分けが特徴です。 戦いに出入りするキャラクターの動きは非常に流動的です。 このショーでは、コメディー効果のアニメーションモーメントはそれほど詳細ではありませんが、それでも常に高い水準を維持しています。

ここでもまた、F / k2のキャラクターデザインはディテールと敬意の両方により素晴らしいものです。 Kuroは非常にすばらしく目を覚ましています（キャストの残りの部分と同じように）ので、彼女のデザインは他の人と完全に一致しています。
